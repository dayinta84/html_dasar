Saat setiap input diberi atribut "name" yang berbeda, setiap pilihan dianggap sebagai kumpulan terpisah, sehingga hanya satu pilihan yang dapat dipilih pada satu waktu

Uji coba 5




Input dengan type "datetime-local" memberikan pengguna opsi untuk memilih tanggal dan waktu dari kalender dan jam yang ditampilkan.
Input dengan type "month" memberi opsi pengguna untuk memilih bulan dan tahun dari dropdown yang disediakan.
Input dengan type "email" memberikan validasi otomatis untuk memastikan bahwa format email yang dimasukkan sesuai.

Uji coba 6




Input dengan tipe "image" menampilkan gambar sebagai tombol submit. Ketika gambar tersebut diklik, posisi klik akan dikirim sebagai bagian dari data formulir.
Input dengan tipe "file" memungkinkan pengguna memilih file dari perangkat lokal mereka.
Input dengan tipe "hidden" tidak ditampilkan pada halaman web tetapi nilainya akan dikirim sebagai bagian dari data formulir saat formulir itu dikirimkan.
Biasanya digunakan untuk menyimpan data yang tidak ingin ditampilkan kepada pengguna.

Uji coba 7



Ketika nilai min pada input tipe number diubah menjadi -50, rentang nilai yang diizinkan menjadi antara -50 dan 10. Pada input tipe range, perubahan nilai min tidak mempengaruhi tampilan atau perilaku langsung dari slider, namun, hal itu memperluas rentang nilai yang diizinkan dalam formulir, sehingga pengguna dapat memilih nilai di luar rentang yang didefinisikan secara awal. Dalam hal ini, pengguna dapat memilih nilai di antara -50 dan 50

Uji coba 8




Input dengan tipe "url" memvalidasi input sebagai URL yang valid.
Input dengan tipe "search" menghasilkan input dengan tombol pencarian yang memungkinkan pengguna melakukan pencarian langsung pada browser.
Input dengan tipe "tel" memvalidasi input sebagai nomor telepon dengan pola yang ditentukan dalam atribut "pattern". Dalam kasus ini, polanya adalah tiga digit, tanda strip, dua digit, tanda strip, dan tiga digit.

Uji coba 9



Input dengan tipe "time" memungkinkan pengguna memilih waktu menggunakan kontrol jam dan menit. Format waktu yang dipilih sesuai dengan preferensi pengaturan lokal pengguna.
Input dengan tipe "week" memungkinkan pengguna memilih minggu dalam tahun menggunakan kontrol minggu. Format yang dipilih biasanya tergantung pada pengaturan lokal pengguna.
